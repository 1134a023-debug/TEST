============= 程式碼白話文對照表 =============

這份文件可以當成你的防呆單字卡，以後看到 C# 程式碼就可以拿出來對照：

【 using UnityEngine.UI; / using UnityEngine.SceneManagement; 】
- 白話文：借用特殊工具包。
- 功用：告訴系統我們要用 UI (例如 Text, Image) 或是 場景切換 (LoadScene)。沒這行系統會看不懂那些單字。

【 public 】
- 白話文：「公開的」變數或功能。
- 功用：只要加了這個字，Unity 右邊的 Inspector 屬性面板就會挖出一個洞，讓你能用滑鼠把圖片、文字框拖進去，或是讓按鈕能選到你的功能。

【 void (例如 public void StartGame) 】
- 白話文：執行動作 (函式)。
- 功用：void 代表這是一個「只負責做事，不用算數學回傳解答」的行動。我們把換場景寫在這個動作裡，讓點擊按鈕時觸發它。

【 int (例如 int targetNumber) 】
- 白話文：整數。
- 功用：用來宣告這個變數只能裝「整數」。我們用它來存炸彈密碼和剩餘猜測次數。

【 Start() 】
- 白話文：開局自動執行的功能。
- 功用：這是 Unity 規定的名稱。只要遊戲一開始，或是剛切換到新場景，裡面的程式碼就會「自動跑一次」（例如開局就抽密碼、或一到結局就去檢查輸贏）。

【 Random.Range(1, 100) 】
- 白話文：隨機抽數字。
- 功用：第一關按 Play 時，它會在 1~99 之間幫我們抽出一個隨機數字，當作這局真正的炸彈密碼。

【 SceneManager.LoadScene(場景名字) 】
- 白話文：場景切換器。
- 功用：用來載入另一個畫面。我們用它從開始畫面載入到計算機，再從計算機載入結局畫面。

【 int.Parse(字串) 】
- 白話文：把純文字轉回數學數字。
- 功用：因為按鈕疊上去的是「文字」"75"，程式不能拿文字跟數學密碼比大小。這行負責把你打的字變回「能比大小的數學整數 75」，才能知道有沒有猜中。

【 PlayerPrefs.SetInt("GameResult", 1) 】
- 白話文：跨關卡記憶卡 (存檔)。
- 功用：Unity 一換場景就會全部失憶，所以不管贏或輸，都必須靠這行把結果存進硬碟裡保留！(這份作業：猜中存 1 代表贏，猜錯炸彈爆炸存 0 代表輸)。

【 PlayerPrefs.GetInt("GameResult", 0) 】
- 白話文：跨關卡記憶卡 (讀檔)。
- 功用：到了結局場景時，用這行翻開硬碟記憶卡，看看裡面存的是 1 還是 0。藉此判斷應該要顯示過關的「打勾圖片」還是失敗的「爆炸圖片」。
