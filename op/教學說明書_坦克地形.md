# 坦克移動與地形地物 (HW) 製作說明書

這份說明書將指引你完成 **「射擊遊戲設計05 地形地物」** 與 **「射擊遊戲設計06 Character移動控制」** 的作業。
我們已經幫你把坦克控制的 C# 腳本 `Player2.cs` 建立在 `op` 資料夾中。請依照以下步驟在 Unity 中完成設定。

---

## ⛰️ 第一階段：地形地物製作 (PDF 05 內容)

1. **建立 Terrain (地形)**：
   - 在 Unity 的 Hierarchy 視窗中 `點擊右鍵 -> 3D Object -> Terrain`。
   - 這會建立一個大片的地形區塊，並在專案中自動生成 `New Terrain` 資料。

2. **地形起伏塑造 (Raise/Lower Terrain)**：
   - 點選 Terrain 物件，在 Inspector 中找到 **Terrain 畫筆工具** (第二個圖示)。
   - 下拉選單選擇 **Raise or Lower Terrain**。
   - 選擇一個筆刷（Brushes），調整 **Brush Size** (畫筆大小) 與 **Opacity** (不透明度/強度)。
   - **在 Scene 視窗中塗抹**：按住滑鼠左鍵可以拉高地形（做成山丘），按住 `SHIFT` 鍵再點左鍵可以降低地形（做出低谷）。
   - 可用選單中的 **Smooth Height** 撫平過於陡峭的邊緣，讓山丘更平滑，方便坦克行駛。

3. **繪製地形紋理貼圖 (Paint Texture)**：
   - 點選畫筆工具，下拉選單選擇 **Paint Texture**。
   - 點擊 **Edit Terrain Layers... -> Create Layer...**，選擇適合的草地或泥土貼圖（如 `GrassHill`）。
   - **基礎層**：第一個加入的層會自動鋪滿整個地形。
   - **細節層**：再次點擊 `Create Layer...` 加入第二個層（如岩石或泥地貼圖），點選它並在場景中刷出道路或山壁的顏色。

4. **設定天空盒與霧氣 (Skybox & Fog)**：
   - **Skybox**：點選上方選單 `Window -> Rendering -> Lighting` (舊版為 `Window -> Lighting -> Settings`)。
   - 在 **Environment** 分頁中，點選 **Skybox Material**，選擇一個適合的天空盒材質。
   - **Fog**：在同一個 Lighting 設定視窗下方，勾選 **Fog**。
     - 調整 **Color** 設定霧的顏色，並修改 **Mode** (通常選 Linear 比較好調整) 與 **Density / Start-End 距離** 來呈現朦朧的戰場效果。

## 🚜 第二階段：坦克車物件結構建置 (PDF 06 內容)

依照 PDF 上的「方塊世界坦克車」結構，我們使用 3D 基礎幾何體來拼接一台坦克：

1. **建立坦克底座 (TankBody)**：
   - 在 Hierarchy 中 `右鍵 -> 3D Object -> Cube`，命名為 `Tank11` (或 `TankBody`)。
   - 調整 Scale，使其看起來像一個長方形的坦克車身。
   
2. **建立坦克砲塔 (TankTop)**：
   - 在 `TankBody` 物件上 `右鍵 -> 3D Object -> Cube` (作為子物件)，命名為 `Tank11_Top` (或 `TankTop`)。
   - 調整大小並移動到坦克底座的上方。因為是子物件，砲塔旋轉時底座不會旋轉。

3. **建立坦克砲管 (TankEmitter)**：
   - 在 `TankTop` 物件上 `右鍵 -> 3D Object -> Cylinder` (作為子物件)，命名為 `Tank11_Can` (或 `TankEmitter`)。
   - 旋轉並拉伸它，使其看起來像一根朝前的砲管。
   - **重要：** 確保砲管的 Pivot (旋轉中心) 在與砲塔相連的根部，這樣上下旋轉時才會正常（或者可以使用一個空的 GameObject 包裹它來做為旋轉錨點）。

## ⚙️ 第三階段：物理與腳本掛載 (PDF 06 內容)

1. **加入物理碰撞與重力**：
   - 點選坦克本體最外層物件（底座 `TankBody`）。
   - 點擊 `Add Component -> Physics -> Rigidbody` (鋼體)。
     - 確保勾選 `Use Gravity` (使用重力)。
   - 點擊 `Add Component -> Physics -> Box Collider` (碰撞器)，並點擊 `Edit Collider` 調整大小，使其完美包裹住底座。
   - 這樣做能讓坦克受到重力影響，並能夠**沿著地形的高低起伏上下行駛**而不穿模。

2. **掛載控制腳本 (`Player2.cs`)**：
   - 將本專案 `op` 資料夾中的 `Player2.cs` 腳本拖曳掛載至 `TankBody` 物件上。
   - 點選 `TankBody`，在 Inspector 的 `Player 2 (Script)` 元件中進行欄位拖曳指定：
     - **Tank Body** 欄位：把 `TankBody` 物件拉進去。
     - **Tank Top** 欄位：把 `Tank11_Top` (砲塔) 物件拉進去。
     - **Tank Emitter** 欄位：把 `Tank11_Can` (砲管) 物件拉進去。
     - **Hp** 設為 3。

## 🎮 第四階段：控制說明

播放遊戲（點擊 Play）後即可使用鍵盤控制坦克運作：

| 按鍵 | 控制對象 | 動作說明 |
| :---: | :---: | :---: |
| **W / S** | 坦克底座 | 前進 / 後退 (沿著坦克 transform.forward 方向) |
| **A / D** | 坦克底座 | 左轉彎 / 右轉彎 (旋轉底座 Y 軸) |
| **Q / E** | 砲塔 | 左旋轉 / 右旋轉 (旋轉砲塔 Y 軸) |
| **R / F** | 砲管 | 砲管抬高 / 砲管降低 (旋轉砲管 X 軸，限制為 -10 到 45 度) |

## 📝 繳交檢查清單
- [ ] 地形有起伏山丘，且塗抹了兩種以上的貼圖紋理。
- [ ] 坦克由底座、砲塔、砲管組成，層級結構正確（砲塔與砲管為子物件）。
- [ ] 坦克本體具有 Rigidbody 與 Box Collider，能在地形上靠重力平穩行駛。
- [ ] 鍵盤控制測試：坦克移動、轉向，砲塔水平旋轉，砲管垂直俯仰限制均運作正常。
