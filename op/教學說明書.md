# HW0513 彈珠台 (Pinball) 製作說明書

這份說明書將指引你完成使用 Unity 3D Object 製作包含關節碰撞的彈珠台作業。
所有需要的 C# 程式碼已經幫你放在這個資料夾中，請直接將它們拖曳到對應的物件上即可。

---

## 🛠️ 第一階段：場景與基礎物件建置

1. **建立彈珠球檯 (Board)**：
   - `Hierarchy -> 右鍵 -> 3D Object -> Cube`，命名為 `Board`。
   - 調整 `Scale` (例如 X: 10, Y: 0.5, Z: 15)，並將 `Rotation X` 設為 `10` 產生傾斜。
   - 四周建立 4 個 Cube 當作**圍牆 (Walls)**。
2. **建立彈珠 (Marble)**：
   - 建立一個 Sphere 命名為 `Marble`。
   - `Add Component -> Rigidbody` (讓它能往下滾動)。
   - 將它的 Tag 設定為 `Player` (沒有的話請新增)。
   - **掛載程式碼**：將 `MarbleCollision.cs` 拖曳給它。
3. **建立黑洞 (Black Hole) (進階挑戰)**：
   - 在球檯最下方建立一個透明 Cube，命名為 `BlackHole`。
   - 勾選 Collider 的 `Is Trigger`。
   - 將它的 Tag 設定為 `BlackHole`。

---

## 🔗 第二階段：關節碰撞設定 (Joints)

1. **左右球門 / 擋板 (Flippers)**：
   - 建立兩個長條形 Cube，命名為 `LeftFlipper` 與 `RightFlipper`。
   - 加入 **Rigidbody**，**取消勾選** `Is Kinematic`。
   - 加入 **Hinge Joint**：
     - **Axis** 設為 `0, 1, 0` (Y 軸)。
     - 勾選 **Use Limits**，設定 `Min: -45, Max: 45`。
     - 勾選 **Use Spring**，設定 Spring 為 `1000`，Damper 為 `10`，Target Position 為 `0`。
   - **掛載程式碼**：將 `FlipperController.cs` 拖曳給它們。在 Inspector 中，左擋板勾選 `Is Left Flipper`，右擋板不勾選。
2. **旋轉圓形障礙物 (Spinning Bumpers)**：
   - 建立 Cylinder，加入 **Hinge Joint**。
   - 勾選 **Use Motor**，設定 Target Velocity `300`，Force `100`。
3. **發射器 (Launcher)**：
   - 在右下方建立發射通道，並建立一個 Cube 命名為 `Plunger`。
   - 幫 `Plunger` 加上 Rigidbody，取消勾選 Use Gravity，並勾選 Freeze Rotation (X, Y, Z) 與 Freeze Position (X, Y)。
   - **掛載程式碼**：將 `Launcher.cs` 拖曳給 `Plunger`。在 Inspector 欄位中把彈珠 `Marble` 拖進去。

---

## 💻 第三階段：UI 介面與計分道具

1. **計分道具 (Score Items) (進階挑戰)**：
   - 建立幾個小圓柱體當作得分道具。
   - 將它們的 Tag 設定為 `ScoreItem`。
2. **UI 介面**：
   - `右鍵 -> UI -> Text` 命名為 `ScoreText`。
   - `右鍵 -> UI -> Button` 分別建立「開始遊戲」與「結束遊戲」按鈕。
3. **遊戲管理器**：
   - 建立一個空的 GameObject 命名為 `GameManager`。
   - **掛載程式碼**：將 `GameManager.cs` 拖曳給它。
   - 在 Inspector 中把剛才建立的 `ScoreText` 拖曳進對應欄位。
   - 在兩個 Button 的 `On Click ()` 事件中，把 `GameManager` 拉進去，分別選擇 `GameManager.RestartGame` 和 `GameManager.QuitGame`。

---

2. 儲存場景 (`Ctrl + S`)。
