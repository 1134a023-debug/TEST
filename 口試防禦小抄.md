# 🎮 Unity 遊戲設計口試防禦小抄

這份小抄幫你把程式碼翻譯成「給老師聽的白話文」。請熟記底下的**粗體字關鍵字**！

---

## 🎳 專案一：保齡球 (Bowling)

### 1. 丟球前：讓球跟著拉桿左右移動
```csharp
if (!hasThrown && angleSlider != null) {
    Vector3 pos = transform.position;
    pos.x = angleSlider.value;
    transform.position = pos;
}
```
*   **在幹嘛？** 讓球的 X 軸（左右）跟著畫面上的拉桿移動。
*   **一行行解釋：**
    1. `if (!hasThrown)`：如果**還沒丟出球**才執行，避免丟出去後還能隔空控球。
    2. `Vector3 pos = transform.position`：先拿一個叫做 `pos` 的箱子，裝進球目前的座標。
    3. `pos.x = angleSlider.value`：把 `X軸` 座標換成**「拉桿現在的數字」**。
    4. `transform.position = pos`：把改好的座標塞回球身上，完成移動。

### 2. 按下按鈕：把球丟出去
```csharp
public void ThrowBall() {
    if (!hasThrown) {
        angleSlider.gameObject.SetActive(false);
        rb.AddForce(Vector3.forward * throwForce);
        hasThrown = true;
    }
}
```
*   **在幹嘛？** 按下按鈕後，發射保齡球。
*   **一行行解釋：**
    1. `public void ThrowBall()`：設為公開功能，按鈕才找得到。
    2. `SetActive(false)`：要丟球了，所以把畫面上的拉桿**隱藏起來**。
    3. **`rb.AddForce(...)` (這行必考！)**：呼叫剛體 (`rb`)，給它一個**往前 (`Vector3.forward`) 的推力 (`AddForce`)**。
    4. `hasThrown = true`：記錄已經丟球了，防止玩家狂按按鈕重複加速。

## 🕹️ 專案二：彈珠台 (Pinball)

### 1. 擋板 (Flipper) 的揮動
```csharp
if (isLeftFlipper && Input.GetKey(KeyCode.LeftArrow)) { 
    isPressed = true; 
}
if (isPressed) { 
    springConfig.targetPosition = -45f; 
} else {
    springConfig.targetPosition = 0f;
}
hinge.spring = springConfig;
```
*   **在幹嘛？** 按下左鍵時，左擋板會往上揮；放開時掉下來。
*   **一行行解釋：**
    1. **`Input.GetKey` (這行必考！)**：偵測玩家有沒有「按住」鍵盤的**左方向鍵**。
    2. 如果有按，就把彈簧 (`spring`) 的**目標角度 (`targetPosition`) 設為 -45 度**，擋板就會揮上去。
    3. 如果沒按，就把目標角度設為 0 度，擋板就會因為彈簧的效果自己掉回原位。
    4. `hinge.spring = springConfig`：把設定好的彈簧參數，正式套用到關節上。

### 2. 發射器 (Launcher) 蓄力發射
```csharp
// 按住空白鍵時 (蓄力)
if (Input.GetKey(KeyCode.Space)) {
    currentPower += chargeRate * Time.deltaTime;
}

// 放開空白鍵時 (發射)
if (Input.GetKeyUp(KeyCode.Space)) {
    marbleRb.AddForce(transform.forward * currentPower);
}
```
*   **在幹嘛？** 按住空白鍵累積力量，放開時把彈珠射出去。
*   **一行行解釋：**
    1. `Input.GetKey`：當**按住**空白鍵時，力量 (`currentPower`) 會隨著時間慢慢增加。
    2. `Input.GetKeyUp`：當**放開**空白鍵的瞬間，觸發發射。
    3. **`AddForce` (這行必考！)**：用剛才累積好的力量 (`currentPower`)，往發射器的**正前方 (`transform.forward`)** 用力推彈珠一把。

### 3. 碰到柱子計分
```csharp
void OnCollisionEnter(Collision collision) {
    if (collision.gameObject.CompareTag("ScoreItem")) {
        gm.AddScore(100); 
    }
}
```
*   **在幹嘛？** 彈珠撞到障礙物時加分。
*   **一行行解釋：**
    1. **`OnCollisionEnter` (這行必考！)**：這是 Unity 內建的「**碰撞偵測**」功能，只要這顆彈珠撞到東西就會自動觸發。
    2. `CompareTag("ScoreItem")`：檢查撞到的那個東西，它身上貼的**標籤 (Tag)** 是不是叫做 `ScoreItem`。
    3. `AddScore(100)`：如果是的話，就呼叫 GameManager 加 100 分。
