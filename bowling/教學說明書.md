# Unity 3D 碰撞遊戲教學說明書：保齡球 (HW0429)

本說明書將帶領你從零開始，使用 Unity 3D 製作一款基礎的保齡球碰撞遊戲。內容涵蓋場景建模、物理設定、程式腳本、UI 介面，以及進階的計分與操作控制功能。

---

## 階段一：專案建立與場景建模 (3D Objects)

### 1. 建立專案
1. 開啟 Unity Hub，點擊 **New Project**。
2. 選擇 **3D Core** 模板。
3. 專案名稱命名為 `BowlingGame_HW0429`，點擊 **Create Project**。

### 1.5 建立與儲存場景 (Scene)
Unity 開啟新專案時預設會提供一個 `SampleScene`。我們可以直接使用它，但為了養成好習慣，請先把它重新命名並存檔：
1. 在左上角功能表點擊 `File` -> `Save As...`。
2. 進入 `Scenes` 資料夾，將檔案命名為 `BowlingScene` 並存檔。
3. 這時候左上方 Hierarchy 面板頂端就會顯示 `BowlingScene`，代表我們現在是在正確的場景裡面工作囉！

### 2. 製作保齡球跑道 (Lane)
1. 在 Hierarchy 面板點擊右鍵 -> `3D Object` -> `Cube`。
2. 將其重新命名為 `Lane` (跑道)。
3. 在 Inspector 面板，將 Transform 的 Scale 調整為 `X: 5, Y: 0.1, Z: 20`。
4. 為了防止球掉出去，可以在兩側建立擋板 (Gutter)：複製 `Lane`，調整 Scale 為 `X: 0.5, Y: 0.5, Z: 20` 並移動到跑道兩側。

### 3. 製作保齡球 (Bowling Ball)
1. Hierarchy 右鍵 -> `3D Object` -> `Sphere`。
2. 重新命名為 `Ball`。
3. 將其移動至跑道起點 (例如 Position `X: 0, Y: 0.5, Z: -8`)。

### 4. 製作保齡球瓶 (Pins)
1. Hierarchy 右鍵 -> `3D Object` -> `Cylinder`。
2. 重新命名為 `Pin`，並將 Transform 的 Scale 調整為 `X: 0.5, Y: 1, Z: 0.5`。
3. 將 `Pin` 移動到跑道末端。
4. 按下 `Ctrl + D` (Mac 為 `Cmd + D`) 複製多個球瓶，並排成三角形陣列 (1-2-3-4 排列)。

## 階段二：物理與碰撞設定 (Physics)

為了讓球能撞擊瓶子，我們必須為物件加上剛體 (Rigidbody)。

### 1. 保齡球的剛體設定
1. 選取 `Ball`，在 Inspector 點擊 **Add Component**。
2. 搜尋並加入 `Rigidbody`。
3. 在 Rigidbody 屬性中，可以將 **Mass** (質量) 設為 `5`，讓球比較有重量感。
4. 確認 `Sphere Collider` 已經存在。

### 2. 球瓶的剛體設定
1. 同時選取所有 `Pin` 物件。
2. 點擊 **Add Component**，加入 `Rigidbody`。
3. 將 **Mass** 設為 `1`。
4. 為了讓瓶子被撞擊時彈開效果更好，可以建立物理材質 (Physic Material)：
   - 在 Project 面板右鍵 -> `Create` -> `Physic Material`，命名為 `BouncyMaterial`。
   - 將 `Bounciness` 設為 `0.5`。
   - 將這個材質拖曳到所有 `Pin` 的 Capsule Collider 中的 `Material` 欄位。

## 階段三：腳本與程式部分 (Scripts)

### 1. 讓球往前滾動
1. 在 Project 面板右鍵建立資料夾 `Scripts`。
2. 進入資料夾，右鍵 -> `Create` -> `C# Script`，命名為 `BallController`。
3. 將腳本拖曳到 `Ball` 物件上，打開腳本並輸入以下程式碼：

```csharp
using UnityEngine;

public class BallController : MonoBehaviour
{
    public float throwForce = 1500f; // 投球力道
    private Rigidbody rb;
    private bool hasThrown = false;

    void Start()
    {
        rb = GetComponent<Rigidbody>();
    }

    // 將投球功能獨立出來，讓 UI 可以呼叫
    public void ThrowBall()
    {
        if (!hasThrown)
        {
            // 在 Z 軸方向施加向前的推力
            rb.AddForce(Vector3.forward * throwForce);
            hasThrown = true;
        }
    }
}
```

## 階段四：UI 介面製作 (開始與結束遊戲)

### 1. 建立 UI 畫布
1. Hierarchy 右鍵 -> `UI` -> `Canvas`。
2. 選取 Canvas，在 Inspector 中將 UI Scale Mode 設為 `Scale With Screen Size`。

### 2. 新增開始遊戲按鈕
1. Hierarchy 右鍵 -> `UI` -> `Button - TextMeshPro` (如果是舊版則選 Button)。
2. 命名為 `StartButton`，將裡面的文字改為 `開始遊戲`。
3. 在 `StartButton` 的 Inspector 找到 `On Click ()` 區塊，點擊 `+` 號。
4. 將場景上的 `Ball` 拖曳進去，下拉選單選擇 `BallController` -> `ThrowBall()`。

### 3. 新增結束遊戲按鈕
1. 複製按鈕，命名為 `QuitButton`，文字改為 `結束遊戲`。
2. 新增一個 C# Script 叫 `GameManager`，放在空的 GameObject (命名為 GameManager) 上：

```csharp
using UnityEngine;

public class GameManager : MonoBehaviour
{
    public void QuitGame()
    {
        Debug.Log("遊戲結束！");
        Application.Quit();
    }
}
```
3. 將 `QuitButton` 的 On Click 事件綁定到 `GameManager` 的 `QuitGame()` 方法。

## 階段五：進階功能 (計分與角度調整)

### 1. 球的移動角度控制 (Slider UI)
1. Hierarchy 右鍵 -> `UI` -> `Slider`。
2. 將 Slider 的位置調整到畫面下方，並在 Inspector 中設定 Min Value = `-5`，Max Value = `5`。
3. 修改 `BallController` 程式碼，增加與 Slider 的連動：

```csharp
using UnityEngine;
using UnityEngine.UI;

public class BallController : MonoBehaviour
{
    public float throwForce = 1500f;
    private Rigidbody rb;
    private bool hasThrown = false;
    
    [Header("UI 連結")]
    public Slider angleSlider; // 拖入畫面上的 Slider

    void Start()
    {
        rb = GetComponent<Rigidbody>();
    }

    void Update()
    {
        // 如果還沒丟球，球的 X 軸位置跟隨 Slider 的值移動
        if (!hasThrown && angleSlider != null)
        {
            Vector3 pos = transform.position;
            pos.x = angleSlider.value;
            transform.position = pos;
        }
    }

    public void ThrowBall()
    {
        if (!hasThrown)
        {
            // 將 Slider 隱藏或停用
            if (angleSlider != null) angleSlider.gameObject.SetActive(false);
            
            rb.AddForce(Vector3.forward * throwForce);
            hasThrown = true;
        }
    }
}
```

### 2. 簡易計分功能
1. 在瓶子後面放置一個透明的「偵測區」(Trigger Collider)。當瓶子倒下進入該區域時，計分加一。
2. 或者，在 `Pin` 物件上掛載腳本，當瓶子的傾斜角度大於特定值 (如 45 度) 時，將分數傳給 `GameManager` 並顯示在 Text 介面上。
